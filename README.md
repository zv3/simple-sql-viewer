# SQL Viewer

## Overview 

The aim of this project is to build a web app to run and visualize SQL queries from a browser.

[DEMO](https://vue3-sql-viewer.netlify.com)

## Technology stack

- Vue3
- TailwindCSS + Flowbite
- Pinia
- Vite
- Monaco Editor
